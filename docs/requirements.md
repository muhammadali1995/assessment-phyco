## 2. API Keys & Tokens

### 2.1 OpenAI API

| Item | Details |
|---|---|
| **Key name** | `OPENAI_API_KEY` |
| **Where to get** | https://platform.openai.com/api-keys |
| **Model required** | `gpt-4o` (Structured Outputs support required) |
| **Billing** | Pay-as-you-go — ensure billing is active and usage limits are set |
| **Permissions** | Chat Completions with JSON mode / Structured Outputs |

### 2.2 FluentCRM REST API

| Item | Details |
|---|---|
| **Key name** | `FLUENTCRM_API_URL` |
| **Format** | `https://<wordpress-domain>/wp-json/fluent-crm/v2` |
| **Auth keys** | `FLUENTCRM_API_KEY` and `FLUENTCRM_API_SECRET` |
| **Where to get** | WordPress Admin → FluentCRM → Settings → REST API |
| **Permissions** | Full API access (contacts, tags, custom fields, lists) |

### 2.3 Fluent Forms Webhook

| Item | Details |
|---|---|
| **Key name** | `WEBHOOK_BEARER_TOKEN` |
| **Purpose** | Server-to-server auth for incoming form submissions |
| **Set in** | Fluent Forms → Form Settings → Webhooks → Header: `Authorization: Bearer <token>` |
| **Generated by** | You — generate a strong random token (64+ chars) |

### 2.4 Payment Provider Webhook

| Item | Details |
|---|---|
| **Key name** | `PAYMENT_WEBHOOK_SECRET` |
| **Purpose** | HMAC signature verification for payment webhooks |
| **Provider** | TBD from client (Stripe, PayPal, etc.) |
| **If Stripe** | `STRIPE_WEBHOOK_SIGNING_SECRET` from Stripe Dashboard → Developers → Webhooks |
| **If Stripe** | Also needs `STRIPE_SECRET_KEY` for payment verification |

### 2.5 DigitalOcean Spaces (S3-compatible)

| Item | Details |
|---|---|
| **Key names** | `DO_SPACES_KEY`, `DO_SPACES_SECRET` |
| **Also needed** | `DO_SPACES_BUCKET`, `DO_SPACES_REGION`, `DO_SPACES_ENDPOINT` |
| **Where to get** | DigitalOcean Console → API → Spaces Keys |
| **Purpose** | PDF storage + pre-signed URL generation |

### 2.6 Admin Portal Auth

| Item | Details |
|---|---|
| **Key name** | `JWT_SECRET` |
| **Purpose** | Signing JWT tokens for admin portal authentication |
| **Generated by** | You — generate a strong random secret (64+ chars) |
| **Also needed** | `ADMIN_DEFAULT_EMAIL`, `ADMIN_DEFAULT_PASSWORD` (initial seed) |

## 3. Database Credentials

| Variable | Description |
|---|---|
| `POSTGRES_HOST` | Database host (e.g., `db` in Docker, `localhost` in dev) |
| `POSTGRES_PORT` | Default `5432` |
| `POSTGRES_DB` | Database name (e.g., `assessment_platform`) |
| `POSTGRES_USER` | Database user |
| `POSTGRES_PASSWORD` | Strong password |
| `REDIS_HOST` | Redis host (e.g., `redis` in Docker) |
| `REDIS_PORT` | Default `6379` |
| `REDIS_PASSWORD` | Optional but recommended for production |

---

## 4. SSL / Domain

| Item | Details |
|---|---|
| **Domain** | A domain/subdomain pointing to the DigitalOcean Droplet (e.g., `api.saferelate.com`) |
| **SSL** | Let's Encrypt via Certbot — auto-provisioned in Docker setup |
| **Variable** | `APP_DOMAIN` — the public-facing domain |
| `APP_PORT` | Internal Express port (e.g., `3000`) |

---

## 5. Application Config Variables

| Variable | Default | Description |
|---|---|---|
| `NODE_ENV` | `production` | Environment mode |
| `REPORT_URL_EXPIRY_DAYS` | `7` | Pre-signed URL expiry duration |
| `BULLMQ_MAX_RETRIES` | `3` | Max retry attempts for failed jobs |
| `BULLMQ_BACKOFF_DELAY` | `5000` | Initial backoff delay in ms |
| `FULL_REPORT_PRICE` | `5.00` | Price for full report upsell |
| `LOG_LEVEL` | `info` | Logging verbosity |

## 9. Sample `.env` File

```env
# App
NODE_ENV=development
APP_PORT=3000
APP_DOMAIN=http://localhost:3000
LOG_LEVEL=debug

# Database
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DB=assessment_platform
POSTGRES_USER=app_user
POSTGRES_PASSWORD=CHANGE_ME

# Redis
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=

# OpenAI
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

# FluentCRM
FLUENTCRM_API_URL=https://your-wordpress.com/wp-json/fluent-crm/v2
FLUENTCRM_API_KEY=your_api_key
FLUENTCRM_API_SECRET=your_api_secret

# Webhooks
WEBHOOK_BEARER_TOKEN=your_64_char_random_token

# Payment (Stripe example)
STRIPE_SECRET_KEY=sk_live_xxxxxxxx
STRIPE_WEBHOOK_SIGNING_SECRET=whsec_xxxxxxxx

# DigitalOcean Spaces
DO_SPACES_KEY=your_spaces_key
DO_SPACES_SECRET=your_spaces_secret
DO_SPACES_BUCKET=assessment-reports
DO_SPACES_REGION=nyc3
DO_SPACES_ENDPOINT=https://nyc3.digitaloceanspaces.com

# Auth
JWT_SECRET=your_64_char_random_secret
ADMIN_DEFAULT_EMAIL=admin@saferelate.com
ADMIN_DEFAULT_PASSWORD=CHANGE_ME_ON_FIRST_LOGIN

# Reports
REPORT_URL_EXPIRY_DAYS=7
FULL_REPORT_PRICE=5.00

# BullMQ
BULLMQ_MAX_RETRIES=3
BULLMQ_BACKOFF_DELAY=5000
```
